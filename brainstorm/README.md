# Problem Discovery Library

This directory contains research-backed problem opportunities for a one-hour Bridges conference build. Start with [SUMMARY.md](SUMMARY.md), then open the dossier for the top idea that has a real stakeholder available.

```text
brainstorm/
  SUMMARY.md                 ranked summary of every idea
  SELECTION_GUIDE.md         five-minute selection and kill criteria
  SOURCES.md                 cross-portfolio evidence index
  municipal/                 eight municipal opportunities
  public-education/          eight public-education opportunities
```

Each dossier contains problem evidence, a product hypothesis, a strict one-hour build boundary, a five-step demo, a minimal data contract, success measures, safety guardrails, pilot questions, an implementation clock, and authoritative sources.

These are discovery briefs, not findings that a particular agency or school has failed. Validate the workflow and language with affected users before representing any idea as locally proven.
