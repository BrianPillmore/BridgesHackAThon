# Five-Minute Idea Selection Guide

## Minute 0-1: Find the stakeholder

Name the person in the room who feels the problem and can reject a bad assumption. Without that person, prefer M01 311 Signal, M02 SafeHeat, M04 Safe Routes, or M08 Capital Project Watch because the demo can stay grounded in public data.

## Minute 1-2: State the decision

Use this sentence: `For [primary user], the app turns [input] into [one decision or action] so that [measurable operational outcome].`

Reject any pitch containing more than one primary user, more than one core workflow, or a promise to solve the whole system.

## Minute 2-3: Prove the data path

Select exactly one:

- Static fixture checked into the repository.
- Deterministic synthetic data generated by `npm run data:demo`.
- A bounded public API request with a fixture fallback.

Do not depend on a production credential, vendor approval, or data-sharing agreement during the one-hour clock.

## Minute 3-4: Draw the five-click demo

The strongest pattern is:

1. See a prioritized queue.
2. Open one item.
3. Understand why it was surfaced.
4. Take one human-approved action.
5. See the queue or summary improve.

## Minute 4-5: Apply kill criteria

Do not choose the idea when any answer is no:

- Can we keep real resident/student/person data out of the demo?
- Can we deploy without a new backend integration?
- Can we explain every score or recommendation?
- Can we complete the happy path before minute 40?
- Can we describe the product without making legal, medical, safety, or eligibility claims?

## Fast recommendations

| Situation in the room                       | Pick                               |
| ------------------------------------------- | ---------------------------------- |
| Strongest all-around civic demo             | M01 - 311 Signal                   |
| Public-health and climate audience          | M02 - SafeHeat Outreach Planner    |
| Transportation or school-safety audience    | M04 - Safe Routes Near Schools     |
| Accessibility or service-design audience    | M06 - Accessible City Forms        |
| School leadership audience                  | E01 - Attendance Recovery Board    |
| Family engagement audience                  | E05 - Multilingual Family Digest   |
| Special-education operations expert present | E04 - IEP Service Delivery Tracker |
| No stakeholder and only 45 minutes remain   | M06 or E05 with fictional data     |
